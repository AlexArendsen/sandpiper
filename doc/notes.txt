Sandpiper RDBMS File System:
- A minimalist file system that lives in your DB
---

# Single table ER spec:
FILES
ID		INT(11)
NAME		VARCHAR(255)
FNAME		VARCHAR(255)
PARENT		INT(11)
TAGS		TEXT

DIRECTORIES
ID		INT(11)
NAME		VARCHAR(255)
PARENT		INT(11)
TAGS		TEXT

Design
---
RDBMS will emulate the structure of the file system, actual files will be stored in a flat directory


Methods
---
UploadFile(int parentId, String name, ...) -> success integer
CreateFolder(int parentId, String name) -> success integer

DeleteFile(int fileId) -> success integer
DeleteFolder(int folderId) -> success integer

EditFile(int fileId, ...) -> success integer
RenameFile(int fileId, String name) -> success integer
EditFolder(int folderId, ...) -> success integer 

MoveFile(int fileId, int parentId) -> success integer
MoveFolder(int folderId, int parentId) -> success integer

DumpObjects() -> List of names and paths
Search(String query) -> list of results
